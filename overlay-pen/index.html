プライバシーポリシー

最終更新日: 2026-04-01
本プライバシーポリシーは、Android アプリ「最前面ペン | 画面に描ける注釈ツール」（以下「本アプリ」）に適用されます。

1. 概要

本アプリは、画面上に描画や注釈を重ねて表示するためのツールです。本アプリは、ユーザー情報の収集をできる限り行わない設計を基本方針としています。

2. 収集しない情報

現時点で、本アプリは以下の情報を開発者のサーバーへ送信、収集、共有しません。

  • 氏名、住所、電話番号、メールアドレスなどの個人情報
  • アカウント情報
  • 位置情報
  • 連絡先
  • 写真、動画、音声などのメディアファイル
  • 広告 ID
  • 利用状況の解析データ
  • 購入情報

3. 端末内で一時的に扱う情報

本アプリは、描画中に必要な情報を一時的に端末メモリ上で扱います。これには、描いた線、選択した色、線の太さ、不透明度、現在のツール状態などが含まれます。

  • これらの情報は端末外へ送信されません
  • 開発者または第三者へ共有されません
  • 描画セッション終了時に消去されます
  • 現時点では永続保存機能を提供していません

4. 利用する権限と機能

本アプリは、他のアプリの上に描画 UI を表示するために Android のオーバーレイ関連権限を利用します。また、描画用オーバーレイを安定して利用できるよう、フォアグラウンドサービスを使用します。

5. 第三者提供

本アプリは、ユーザー情報を第三者へ販売、共有、提供しません。現時点では広告 SDK、アクセス解析 SDK、外部課金 SDK、クラッシュレポート SDK も組み込んでいません。

6. 子どものプライバシー

本アプリは、13 歳未満の子どもを主たる対象としたサービスではありません。

7. セキュリティ

本アプリは、端末外へのデータ送信を行わない設計により、取り扱う情報を最小限にしています。今後、保存、共有、課金、分析などの機能を追加し、取り扱う情報が変わる場合は、本プライバシーポリシーを更新します。

8. 改定

本プライバシーポリシーは、法令改正、機能追加、運用変更などに応じて改定されることがあります。重要な変更がある場合は、アプリ内、配布ページ、または本公開ページで周知します。
